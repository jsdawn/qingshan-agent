## ✅ 项目规范

### 编码规范
- 所有注释必须使用中文
- 代码本身使用英文（驼峰命名法）
- 禁止使用 `any` 类型，全栈 TypeScript 开发

### 文档规范
- 所有新增文档按日期存放在 `docs/developer/YYYY-MM-DD/` 目录下
- 文档使用 Markdown 格式
- 所有文本内容必须使用中文

### 项目配置
详见项目根目录的 `.cursorrules` 文件，了解完整的项目规范。
